# AIGen
Android App welche Bilder mittels AI(stablediffusion) generieren kann

AIGen is an Android App which can generate AI images via stable-diffussion.

The App makes calls to the backend Image Generation Application from https://github.com/AUTOMATIC1111/stable-diffusion-webui. 
To enable the backend Application as an API we can start it by using the command `python3 launch.py --nowebui`.
